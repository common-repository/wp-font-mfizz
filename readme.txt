Customize your pages and your post with icons of technology and software (for Geeks). This plugin is a Wordpress adaptation for Font Mfizz.

== Description ==

"WP Font Mfizz" is a WordPress plugin that helps you customize your pages and posts with icons of technology and software easily from Wordpress text editor.

== Installation ==

1. Upload 'wp-font-mfizz' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. Select the icon of your choice, you can make use of the filter names.

2. Optionally you can add your preference settings like the size (pixels) or the color (hexadecimal notation).

3. Saves the changes and sees the results on the page or post.
